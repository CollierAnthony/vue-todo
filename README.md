# vue-crash-2021

## Project setup
```
npm install
```

## Database setup
### MongoDB start (You will need docker)
```
docker compose up -d
```

## Development
### Start the Vue client
```
npm run serve
```

### Start the Express.js server
```
cd server
npm run startServer
```

### Compiles and minifies for production
```
npm run build
```
